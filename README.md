# Contexto RU

Игра в стиле `Contexto` на русском языке с интерфейсом на `tkinter` и предобученными русскими эмбеддингами слов.

## Что уже сделано

- модель автоматически скачивается при первом запуске
- игровой словарь автоматически строится из предобученной модели и сохраняется в локальный кэш
- русские слова нормализуются перед сравнением
- близость считается через cosine similarity
- интерфейс сделан на `tkinter`
- лемматизация через `pymorphy3` опциональна и не требуется для первого запуска

## Структура проекта

- `src/contexto_ru/normalization.py`: приведение слова к нижнему регистру, замена `ё -> е`, опциональная лемматизация через `pymorphy3`
- `src/contexto_ru/embeddings.py`: чтение официального файла `.vec.gz`, построение большого кэша русского словаря и вычисление cosine similarity
- `src/contexto_ru/game.py`: расчет рангов слов относительно загаданного слова
- `src/contexto_ru/ui.py`: desktop-интерфейс на `tkinter`
- `src/contexto_ru/main.py`: точка входа

## Установка

Опционально можно установить поддержку лемматизации, если хотите лучше обрабатывать склонения и формы слов:

```powershell
python -m pip install pymorphy3
```

Если `pymorphy3` не ставится из-за конфликта в Windows `Scripts`, его можно пропустить. Игра все равно будет работать с базовой нормализацией.

## Запуск

```powershell
$env:PYTHONPATH="src"
python -m contexto_ru.main
```

## Что происходит при первом запуске

1. Если файла `models/cc.ru.300.vec.gz` нет, приложение скачает его автоматически.
2. Приложение просканирует модель fastText.
3. Из модели будут выбраны русские слова для игрового словаря.
4. Локальный кэш будет сохранен в `models/ru_vocab_cache.pkl`.
5. После этого игра запустится уже с большим словарем.

При следующих запусках будет использоваться уже готовый кэш.

## Как работает ранжирование

Для загаданного слова игра:

1. нормализует слова
2. берет вектор слова из предобученной модели
3. считает cosine similarity между загаданным словом и каждым словом словаря
4. сортирует слова по близости
5. показывает ранг догадки

Чем меньше ранг, тем ближе слово к ответу.

## Почему словарь все равно нужен

У `Contexto` всегда должен быть набор слов, относительно которого считается ранг. Без этого невозможно сказать, что слово находится, например, на `153` месте по близости.

Раньше здесь использовался маленький ручной список слов, и это было слишком ограниченно. Теперь словарь строится автоматически из предобученной русской модели, поэтому игра охватывает намного больше слов.
